## 考察

### 教科書の解答

awk 部分は条件部だけで済む。それを利用すれば以下

```
$ seq -f '2021-01-01 %g days' 0 364 | date -f- +'%w %F' | grep ^2 | awk -F'[- ]' '++n[$3]==3' | cut -f2 -d' '
```
